# Haldeki.com

> Bölgesel ürün dağıtım platformu - Taze ürünler kapınıza gelsin.

## Hızlı Başlangıç

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Dokümantasyonu oluştur
npm run docs:build
```

## Teknoloji Stack

### Frontend
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- TanStack Query (veri yönetimi)
- React Router (routing)

### Backend
- Supabase (PostgreSQL + Auth + Storage)
- Supabase Edge Functions (Deno)
- Row Level Security (RLS)

## Geliştirme

```bash
# Geliştirme sunucusu
npm run dev

# Testler
npm run test

# Lint
npm run lint

# Build
npm run build

# Dokümantasyon
npm run docs:build
```
