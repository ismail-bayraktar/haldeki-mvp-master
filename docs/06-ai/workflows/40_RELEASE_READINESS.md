# Workflow: Release Readiness

## 1) Minimum release checklist
- Kritik user journey çalışıyor mu?
- RBAC/RLS: tenant izolasyonu test edildi mi?
- Payment/checkout akışında hata mesajları net mi?
- Logs/audit trail var mı?
- Rollback planı var mı?

## 2) Test katmanları
- Smoke tests
- Role-based access tests
- Edge-case tests
- Performance sanity

## 3) Çıktı
- Release checklist
- Test plan
- Known issues listesi
